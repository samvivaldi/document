기본 npm 명령어
npm install -g(g는 글로벌, --save-dev 또는 -D 개발dependence, --save 또는 -S 디폴트) 모듈명  <-- 설치
npm update [패키지명]
npm search [검색어]
npm info [패키지명]
$ npm uninstall <package-name>   로컬/개발 패키지 제거
$ npm uninstall -g <package-name>  전역 패키지 제거
